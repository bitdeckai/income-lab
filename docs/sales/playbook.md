# Sales Playbook

## Weekly Rhythm
1. Publish 2 build/progress updates.
2. Send 10 targeted outreach messages.
3. Run 3 discovery calls.
4. Convert at least 1 sponsor to $49+.

## ICP (Ideal Customer Profile)
- Teams shipping ESP32 products.
- Android apps controlling BLE devices.
- Integrators needing Matter production workflows.

## Offer Ladder
- Free: public docs + basic templates
- $15: faster responses
- $49: private packs and scripts
- $199: monthly integration review

## Outreach Channels
- GitHub issues/discussions
- X/Twitter technical thread
- Reddit embedded communities
- Discord/Telegram hardware groups

## Conversion Notes
- Always lead with delivered value and demo.
- Keep CTA direct: sponsor or integration request.
- Link one proof point per message.
